# 2015_Althaus_Epidemics

This repository contains data and `R` code files from the following publication:

- Althaus CL, Low N, Musa EO, Shuaib F, Gsteiger S. (2015) [Ebola virus disease outbreak in Nigeria: Transmission dynamics and rapid control](http://dx.doi.org/10.1016/j.epidem.2015.03.001). Epidemics, 11:80-4.
